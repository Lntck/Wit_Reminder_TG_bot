# Wit_Reminder_TG_bot
A Telegram Bot which serves as a reminder for your meetings, appointments, events etc.

```/start -> add a new reminder```

## Try the bot
You can use the bot under: @Wit_reminder_bot

## Requirements
* telebot, datetime, time, threading, calendar
